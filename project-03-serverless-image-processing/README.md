# 🖼️ Serverless Image Processing System on AWS

This project demonstrates a **serverless event-driven architecture on AWS** where an image upload to an S3 bucket automatically triggers a Lambda function.

The Lambda function processes the event and logs the execution in **Amazon CloudWatch**, proving that the system runs automatically without managing servers.

---

## AWS Services Used

### Amazon S3
- Stores uploaded images
- Generates events when new objects are created
- Triggers the Lambda function automatically

### AWS Lambda
- Executes serverless code when an S3 event occurs
- Processes the uploaded image event

### Amazon CloudWatch
- Stores logs generated by Lambda
- Helps monitor execution and debugging

---

## Architecture

```
User
  ↓
Upload Image
  ↓
Amazon S3 Bucket
  ↓
S3 Event Trigger
  ↓
AWS Lambda Function
  ↓
CloudWatch Logs
```
